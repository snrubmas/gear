## Planned Music System Roles

- Digitakt: sample engine, drum kits, patterns, field-recording and BeatFactory sample playback.
- Alia: modular percussion and rhythmic texture.
- Mother-32 #1: bass / low oscillator voice.
- Mother-32 #2: upper oscillator / counterline voice.
- Mutant Brain: MIDI-to-gate/CV routing for modular.
- nanoKONTROL2: performance control surface.
- `miniman`: Pi 5 music/MIDI learning host for controlling MIDI devices and running MIDI scene tooling.
- BeatFactory: source library for samples, field recordings, and future Digitakt content.

## Studio Workflow Notes

- Preferred musical direction: drones, evolving soundscapes, randomness with musical constraints, and control voltage as a creative language.
- Preferred session structure: one voice, one modulation source, one question at a time, stable listening chain, no pressure to finish tracks.
- Preferred pitch/modulation starting point: Pamela's PRO Workout into one Mother-32 `1V/OCT`, then explore randomness depth, quantization, speed, filter movement, and envelope behavior.
- Useful MATHS experiment: MATHS Channel 1 cycling output into Mother-32 `VCO 1V/OCT` for slow evolving pitch motion.
- Visualization path: modular audio into Tascam Model 16, then USB into Reaper at 48kHz / 24-bit.
- BeatFactory direction: connects to sample library work, field recordings, visualization, human-centered systems, and exploratory sound design.
